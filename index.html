🔗 API Access

Access the CSV files programmatically with our dual-service architecture:

Direct URLs:

# TA-125 Data (Yahoo Finance) https://raw.githubusercontent.com/YOUR_USERNAME/stock-data-automation/main/data/Shazam-Stock-Info-TA125.csv # S&P 500 Data (Polygon.io) https://raw.githubusercontent.com/YOUR_USERNAME/stock-data-automation/main/data/Shazam-Stock-Info-SP500.csv # Status JSON (includes service health) https://raw.githubusercontent.com/YOUR_USERNAME/stock-data-automation/main/data/download_status.json

Python Example:

import pandas as pd import requests # Load TA-125 data (Yahoo Finance source) ta125_url = "https://raw.githubusercontent.com/YOUR_USERNAME/stock-data-automation/main/data/Shazam-Stock-Info-TA125.csv" ta125_df = pd.read_csv(ta125_url) # Load S&P 500 data (Polygon.io source) sp500_url = "https://raw.githubusercontent.com/YOUR_USERNAME/stock-data-automation/main/data/Shazam-Stock-Info-SP500.csv" sp500_df = pd.read_csv(sp500_url) # Check service status status_url = "https://raw.githubusercontent.com/YOUR_USERNAME/stock-data-automation/main/data/download_status.json" status = requests.get(status_url).json() print(f"TA-125 records: {len(ta125_df)} (via {status['services']['ta125_service']})") print(f"S&P 500 records: {len(sp500_df)} (via {status['services']['sp500_service']})") print(f"Polygon.io Health: {status['services']['polygon_health']['status']}")

Service Architecture:

📊 Multi-Service Design: 🇮🇱 TA-125 Stocks → Yahoo Finance API • Better support for .TA symbols • 15 requests/minute rate limit • 25-30 second delays between requests • Handles Israeli market trading days 🇺🇸 S&P 500 Stocks → Polygon.io API • Premium financial data quality • 20 requests/minute rate limit • 12 second delays between requests • Professional-grade US market data 🔄 Automatic Failover: • Each service optimized for its market • Independent rate limiting per service • Comprehensive health monitoring

JavaScript Example:

// Fetch both datasets and service status Promise.all([ fetch('https://raw.githubusercontent.com/YOUR_USERNAME/stock-data-automation/main/data/Shazam-Stock-Info-TA125.csv'), fetch('https://raw.githubusercontent.com/YOUR_USERNAME/stock-data-automation/main/data/Shazam-Stock-Info-SP500.csv'), fetch('https://raw.githubusercontent.com/YOUR_USERNAME/stock-data-automation/main/data/download_status.json') ]).then(async ([ta125Response, sp500Response, statusResponse]) => { const ta125Data = await ta125Response.text(); const sp500Data = await sp500Response.text(); const status = await statusResponse.json(); console.log('TA-125 CSV (Yahoo Finance):', ta125Data.split('\n').length - 1, 'records'); console.log('S&P 500 CSV (Polygon.io):', sp500Data.split('\n').length - 1, 'records'); console.log('Services:', status.services); });
